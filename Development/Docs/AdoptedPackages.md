Why this doc exists: the package library gets fixes and field notes over time. Without this table,
a project silently carries a stale copy and nobody knows which projects to update when a package
gains a fix, or which local fix should fold back into the canonical package.

## Rules

- Log every copy-in here, in the same task that copies it.
- When you fix a bug inside a copied package, note it under Local Modifications and consider
  folding the fix back into the canonical package README or source.
- When adopting, follow the package's own README checklist first.
